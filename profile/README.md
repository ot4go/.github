# Open Tools for Go Programming (ot4go)
Cool little things I've built because I needed them for my projects, and maybe someone else will find them useful too.

## Current Usable Tools

## Miniskin

It started as a simple pre-processor to handle `<%% include: file %%>` and grew into a full workflow tool. Today, it allows me to work directly on a "touchable" mockup that is pre-processed into the final Go template used by the backend, ensuring no design work is lost in transition. I later discovered this is known as Design-Driven Programming ( wow that sound great 😂 ) 



## Closure UI

A collection of custom HTML elements that I've been building as I needed them for my Go application frontends. However, it's pure JS/HTML5—no Go backend required, it works anywhere.



## Want to contribute?
